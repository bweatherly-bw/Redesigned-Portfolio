Brandon's Recipies

Brandon's Recipes is an application that was built using React and Bootstrap. As a user, you are able to search for a recipe that you want to cook, from there, all the ingredients are listed out for you along with an opton for you to save them to yoour favorites. There is also a Healthy Section that only shows the users healthy low calorie and carb recipes, also on this page, the user is also informed of how many calories each meal has.
